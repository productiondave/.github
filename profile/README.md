# productiondave

This organization hosts the various games and libraries I have put together
over the years.

Placeholder for now while I figure out what projects I will migrate here.

## Games

### Z80

Many of not all of these games will play on the Z180-Nouveau too.

- [https://github.com/productiondave/z80-breakout](https://github.com/productiondave/z80-breakout)
- [https://github.com/productiondave/z80-jumpingjack](https://github.com/productiondave/z80-jumpingjack)
- [https://github.com/productiondave/z80-bombsaway](https://github.com/productiondave/z80-bombsaway)
- [https://github.com/productiondave/z80retro-snake](https://github.com/productiondave/z80retro-snake)
  - snake but in tms9918a multicolor mode
- [https://github.com/productiondave/z80retro-chip-8](https://github.com/productiondave/z80retro-chip-8)

### NABU

These games are released for Z80-Retro!, Z180-Nouveau and the NABU.

- [https://github.com/productiondave/nabu-games](https://github.com/productiondave/nabu-games)
  - Tetris, Snake, Invaders
- [https://github.com/productiondave/NABU-LIB](https://github.com/productiondave/NABU-LIB)
  - Support library for nabu-games targeting the NABU
- [https://github.com/productiondave/z80retro-lib](https://github.com/productiondave/z80retro-lib)
  - Support library for nabu-games targeting the Z80-Retro and the Z180-Nouveau

### 6502

These games are released for the Pico-56 platform.

- [https://github.com/productiondave/pico-56-games](https://github.com/productiondave/pico-56-games)
  - Jumping Jack

## LIBCPM

A standard C library for CP/M development with targets for the NABU, Z80-Retro!
and the Z180-Nouveau.

- [https://github.com/productiondave/z80-compiler-kit-libcpm](https://github.com/productiondave/z80-compiler-kit-libcpm)
